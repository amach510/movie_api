Movie API

A web api that allows users to sign up, update information and create a list of favorite movies.

Movie Endpoints

Request URL Response
GET /movies List of all movies
GET /movies/{title} Data about a single movie by title
GET /genres/{genreName} Data about a genre {description} by name
GET /directors/{directorName} Data about a director (bio, birth year, death year)
GET /release/{releaseYear} Data about all the movies by requested year of release

User Endpoints:

Request URL Response
GET /users Return all users
GET /users/[Username] Return data of specific user
POST /users/register Register a new user
PUT /users/{userId}/update Update user information (username)
POST /users/{userId}/favorites/add

Allows users to add a movie to their list of favorites.

Response: "Movie has been added to favorites."
POST /users/{userId}/favorites/remove

Allows users to remove a movie from their list of favorites.

Response: "Movie has been removed from favorites."
DELETE /users/{userId}/deregister

Allows existing users to deregister.

Response: "User email has been removed."
