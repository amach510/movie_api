Movie API

A web api that allows users to sign up, update information and create a list of favorite movies.

Movie Endpoints

Request URL Response
GET /movies List of all movies
GET /movies/{title} Data about a single movie by title
GET /genres/{genreName} Data about a genre {description} by name
GET /release/{releaseYear} Data about a director (bio, birth year, death year)
GET /year Data about all the movies by requested year of release

User Endpoints:

Request URL Response
POST /users/register Register a new user
PUT /users/{userId}/update Update user information (username)
